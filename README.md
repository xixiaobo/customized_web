# customized_web

## 初始化
### 全局安装淘宝相关镜像依赖
```vue
npm install -g cnpm --registry=http://registry.npm.taobao.org 
```
### 1、安装 axios 进行请请访问
```vue
npm install axios -S
```

### 2、安装 iview
```vue
npm install iview --save
```
